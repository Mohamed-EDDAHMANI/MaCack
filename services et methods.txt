1️⃣ Services Utilisateur
== UserService

register(userData) → inscrire un utilisateur

login(credentials) → authentifier + JWT

getProfile(userId) → récupérer profile

updateProfile(userId, data) → modifier profile

deactivateUser(userId) → désactiver compte

getAllUsers(filters) → pour Admin

== ClientService

getOrders(clientId) → lister commandes

createOrder(orderData) → créer commande

trackOrder(orderId) → suivre commande

rateUser(toUserId, ratingData) → noter pâtissière ou livreur

== PatissiereService

getProducts(patissiereId) → lister gâteaux

createProduct(productData) → publier gâteau

updateProduct(productId, data) → modifier gâteau

deleteProduct(productId) → supprimer gâteau

getOrders(patissiereId) → voir commandes reçues

updateOrderStatus(orderId, status) → changer statut

getStats(patissiereId) → statistiques individuelles

== LivreurService

getAvailableDeliveries() → commandes à livrer

proposeDelivery(deliveryId, price) → proposer prix

acceptDelivery(deliveryProposalId) → accepter livraison

updateDeliveryStatus(deliveryId, status) → changer statut

getEarnings(livreurId) → historique gains

== AdminService / SuperAdminService

getAllUsers(filters)

suspendUser(userId)

getAllOrders(filters)

resolveDispute(disputeId, decision)

configureCommission(type, percentage)

getPlatformStats()
